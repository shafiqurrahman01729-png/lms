🎓 কোর্স সম্পর্কিত

প্রথমে একটি অ্যাকাউন্ট তৈরি করুন বা Login করুন। তারপর কোর্স পেজে গিয়ে "এনরোল করুন" বাটনে ক্লিক করুন। বিনামূল্যের কোর্সে সরাসরি এনরোল হবে, পেইড কোর্সের জন্য পেমেন্ট করতে হবে।
এনরোল করার পর আজীবন অ্যাক্সেস পাবেন। আপনি যেকোনো সময় কোর্সটি আবার দেখতে পারবেন।
না, কোর্সের ভিডিও ডাউনলোড করা যাবে না। তবে ইন্টারনেট সংযোগ থাকলে যেকোনো ডিভাইস থেকে দেখতে পারবেন।
কোর্সের সকল লেসন সম্পন্ন করলে আপনার Dashboard থেকে সার্টিফিকেট ডাউনলোড করতে পারবেন।

💳 পেমেন্ট সম্পর্কিত

আমরা bKash, Nagad এবং Rocket এর মাধ্যমে পেমেন্ট গ্রহণ করি। পেমেন্ট করার পর Transaction ID জমা দিন, Admin যাচাই করলে এনরোল সম্পন্ন হবে।
পেমেন্ট জমা দেওয়ার পর Admin যাচাই করতে ১–২৪ ঘণ্টা সময় লাগতে পারে। যাচাই সম্পন্ন হলে আপনার Dashboard এ কোর্সটি দেখা যাবে। দেরি হলে আমাদের সাথে যোগাযোগ করুন।
হ্যাঁ, পেমেন্টের ৭ দিনের মধ্যে এবং কোর্সের ৩০% এর কম দেখলে রিফান্ড পাওয়া যাবে। বিস্তারিত জানতে আমাদের Refund Policy দেখুন।

👤 অ্যাকাউন্ট সম্পর্কিত

Login পেজে "পাসওয়ার্ড ভুলে গেছেন?" লিংকে ক্লিক করুন। আপনার ইমেইলে একটি রিসেট লিংক পাঠানো হবে।
হ্যাঁ, আপনার অ্যাকাউন্ট দিয়ে মোবাইল, ট্যাবলেট বা কম্পিউটার — যেকোনো ডিভাইস থেকে কোর্স দেখতে পারবেন।
🛠
Help Center
ধাপে ধাপে গাইড
নিচের গাইডগুলো অনুসরণ করে নিজেই সমস্যা সমাধান করুন।

🚀 শুরু করার গাইড

  1. অ্যাকাউন্ট তৈরি করুন — Homepage এ গিয়ে "Register" ট্যাবে নাম, ফোন, ইমেইল ও পাসওয়ার্ড দিয়ে নিবন্ধন করুন।
  2. Login করুন — ইমেইল ও পাসওয়ার্ড দিয়ে Login করুন।
  3. কোর্স বেছে নিন — Dashboard এ সব কোর্স দেখতে পাবেন।
  4. এনরোল করুন — পছন্দের কোর্সে "এনরোল করুন" বাটনে ক্লিক করুন।
  5. শেখা শুরু করুন — "চালিয়ে যান" বাটনে ক্লিক করে লেসন শুরু করুন।

💳 পেইড কোর্সে এনরোলের গাইড

  1. কোর্স কার্ডে "ভর্তি হন" বাটনে ক্লিক করুন।
  2. পেমেন্ট মেথড (bKash/Nagad/Rocket) বেছে নিন।
  3. দেখানো নম্বরে Send Money করুন।
  4. Transaction ID (TrxID) ইনপুটে লিখে "পেমেন্ট নিশ্চিত করুন" ক্লিক করুন।
  5. Admin যাচাই করলে কোর্সে অ্যাক্সেস পাবেন।

⭐ রেটিং দেওয়ার গাইড

  1. Dashboard এ enrolled কোর্সে যান।
  2. "⭐ রেটিং দিন" বাটনে ক্লিক করুন।
  3. ১–৫ স্টার বেছে নিন।
  4. ঐচ্ছিক: মতামত লিখুন।
  5. "রেটিং জমা দিন" ক্লিক করুন।

🔑 পাসওয়ার্ড রিসেটের গাইড

  1. Login পেজে "পাসওয়ার্ড ভুলে গেছেন?" লিংকে ক্লিক করুন।
  2. ইমেইল ঠিকানা দিন।
  3. আপনার ইমেইলে আসা লিংকে ক্লিক করুন।
  4. নতুন পাসওয়ার্ড সেট করুন।

📱 Mobile App গাইড

আমাদের প্ল্যাটফর্ম PWA (Progressive Web App) সাপোর্ট করে। মোবাইল ব্রাউজার থেকে "Home Screen এ Add করুন" অপশনে ক্লিক করে App এর মতো ব্যবহার করুন।

  • Android (Chrome): ব্রাউজারের মেনু → "Add to Home screen"
  • iPhone (Safari): Share বাটন → "Add to Home Screen"

🖥 Browser সাপোর্ট

সেরা অভিজ্ঞতার জন্য নিচের browser গুলো ব্যবহার করুন:

  • Google Chrome (সর্বশেষ ভার্সন) — সবচেয়ে ভালো
  • Mozilla Firefox
  • Microsoft Edge
  • Safari (iOS ১৪+)

🚨 জরুরি সমস্যা

নিচের সমস্যাগুলো জরুরি হিসেবে বিবেচনা করা হয়:

  • পেমেন্ট হয়েছে কিন্তু ২৪ ঘণ্টার বেশি সময় পরেও এনরোল হয়নি
  • অ্যাকাউন্টে লগইন করা যাচ্ছে না
  • কোর্সের ভিডিও চলছে না
  • অ্যাকাউন্ট হ্যাক সন্দেহ

জরুরি সমস্যায় সরাসরি WhatsApp বা ফোনে যোগাযোগ করুন।

📋 সাপোর্ট টিকেট

সাধারণ সমস্যার জন্য ইমেইলে টিকেট পাঠান। টিকেটে নিচের তথ্য দিন:

  • আপনার নাম ও রেজিস্ট্রেশন ইমেইল
  • সমস্যার বিবরণ (যতটা সম্ভব বিস্তারিত)
  • Screenshot (যদি থাকে)
  • কখন থেকে সমস্যা হচ্ছে

⏱ প্রতিক্রিয়ার সময়

  • WhatsApp/ফোন: অফিস সময়ে তাৎক্ষণিক
  • ইমেইল: ১–২ কার্যদিবস
  • সোশ্যাল মিডিয়া: ২–৩ কার্যদিবস

🔧 সাধারণ সমস্যার সমাধান

  • ইন্টারনেট সংযোগ চেক করুন
  • পেজ Refresh করুন (Ctrl+R)
  • Browser Cache পরিষ্কার করুন
  • অন্য Browser এ চেষ্টা করুন
  • VPN বন্ধ থাকলে চালু করুন, চালু থাকলে বন্ধ করুন
  • ইমেইল ও পাসওয়ার্ড সঠিক কিনা নিশ্চিত করুন (Caps Lock চেক করুন)
  • "পাসওয়ার্ড ভুলে গেছেন?" দিয়ে রিসেট করুন
  • Browser এ Cookies এবং Cache পরিষ্কার করুন
  • সমস্যা থাকলে আমাদের সাথে যোগাযোগ করুন
  • পেজ Refresh করুন
  • Logout করে আবার Login করুন
  • পেইড কোর্সের ক্ষেত্রে Admin এর যাচাইয়ের জন্য অপেক্ষা করুন
  • ২৪ ঘণ্টার বেশি সময় হলে আমাদের সাথে যোগাযোগ করুন
